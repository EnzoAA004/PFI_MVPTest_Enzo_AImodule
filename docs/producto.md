# Producto

## Nombre de trabajo

MVP de análisis asistido de RM lumbar sagital.

## Objetivo

Construir un prototipo académico que integre segmentación, mediciones geométricas, visualización y salida estructurada editable para asistir la revisión profesional de resonancias magnéticas lumbares sagitales.

## Qué hace

- Carga imágenes médicas de RM lumbar sagital.
- Preprocesa imágenes para uso computacional.
- Segmenta estructuras anatómicas relevantes.
- Calcula mediciones geométricas simples derivadas de máscaras.
- Visualiza máscaras superpuestas sobre la imagen original.
- Exporta resultados estructurados en formato editable.

## Qué no hace

- No diagnostica.
- No clasifica patologías.
- No recomienda tratamientos.
- No reemplaza la revisión profesional.
- No se presenta como software médico certificado.

## Usuario esperado

Profesional o evaluador técnico que revisa resultados generados por el sistema y puede aceptar, corregir, descartar o comentar las mediciones.

## Principio rector

El sistema debe actuar como herramienta de apoyo operativo, manteniendo siempre al profesional humano como responsable de la interpretación final.
